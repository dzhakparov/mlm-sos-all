# README

## setup of the project

The project can be cloned from github.com with:
```
git clone https://github.com/dzhakparov/mlm-sos-all.git
```

Code is intended to be run from an ```Apptainer``` container to use all resources from the underlying server or system
and to be shareable and executable on all systems. See details in the Apptainer-part of the help-section (Sphinx)
